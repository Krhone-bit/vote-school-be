# vote-school-be

# Requirements

  - Java v17
  - Spring v3.1.4
  - Maven

# Setup
Install dependencies
```bash
mvn clean install
```
Run project
```bash
mvn spring-boot:run
```
# Urls

```url
http://localhost:8081
```
